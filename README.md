# Analyze & Select Data Structure - SLO #5

## What it does
This program creates a simple GUI that lets the user enter numbers and manage them using a **LinkedList**.  
The user can:
- Add numbers (up to 15, no duplicates allowed)
- View the original order
- Sort the list
- Shuffle the list
- Reverse the list

The results are displayed in a text area inside the window.

## How to run it
1. Open the project in your Java IDE.
2. Compile the `Miller_AnalyzeAndSelectDataStructureSLO5.java` file.
3. Run the program.
4. A window will appear where you can enter numbers and use the buttons to manage the list.
